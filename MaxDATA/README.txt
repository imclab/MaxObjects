
The pg.data library is a collection of externals for Max 5 that can be useful for the management, the transformation and the generation of data. Source codes are available and can be useful for the Max new developers.

Objects :

pg.average : Get the moving average
pg.chaos : Chaotic attractors.
pg.distance : Get the distances between ints, floats or lists.
pg.gaussian : Normal distribution generator.
pg.rlg : Random list generator.
pg.scale : A scale object with autoscale mode and boundary mode.
pg.smooth : Smooth the values.
pg.statistic : Get the statisics of a vector

Please report bugs. Other objects will follow!
